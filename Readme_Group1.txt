# photography_app

To run:
1. Clone the repository
2. run `flutter create .` from the root folder
3. Open the mac simulator app or android studio emulator and run `flutter run`

To setup API Keys create a .env file with the following, replacing urls with the api keys from the websites:
```
WEATHER_API_KEY=https://www.weatherapi.com/
OPEN_WEATHER_MAP_API_KEY=https://openweathermap.org/api
LOCATIONIQ_API_KEY=https://locationiq.com/geocoding
```
